# Aditya Menon — Portfolio

Interactive one-page portfolio site highlighting research experience in stem cell biology, bioinformatics, and toxicology.

Built as a single self-contained `index.html` — no build step, no dependencies.

## View locally
Open `index.html` in any browser.

## Live site
Published with GitHub Pages: https://adityamenon09.github.io/portfolio/

## Deploy (GitHub Pages)
This folder is already a git repository with an initial commit on the `main` branch.

1. Create a new empty repository named `portfolio` on GitHub (do not add a README).
2. Connect it and push:
   ```bash
   git remote add origin https://github.com/AdityaMenon09/portfolio.git
   git push -u origin main
   ```
3. On GitHub open **Settings → Pages**, set **Source** to `Deploy from a branch`, pick branch `main` and folder `/ (root)`, then save.
4. The site goes live at https://adityamenon09.github.io/portfolio/ within a minute or two.

Future updates: edit `index.html`, then `git add -A && git commit -m "update" && git push`.

## Deploy (no account, instant)
